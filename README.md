## Getting Started

Read about page of theme demo [tsr-page](https://iam-tsr.github.io/tsr-page/) all the editable instruction are mentioned there in detail.

Below are the setups for installation and setup

### Pre-requisite

```bash
"engines": {
  "node": ">=18.0.0",
  "npm": ">=8.11.0"
}
```

### Installation

1. **Import this template**

![alt text](repo-images/instruct-template.png)

2. **Clone your repo**

```bash
git clone __your-repo-name___
```

3. **Package installation**

```bash
npm ci
```

4. **LFG!**
```
npm run dev
```

## Why you should use it?

- Beginner friendly (Node.js)
- Language: HTML, CSS/Liquid, JS
- Clean packaging
- Minimal, modern UI
- One cmd setup
